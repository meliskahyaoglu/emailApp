# Email Account Generator (Java OOP Project)

## 📌 Overview
This project is a console-based Java application that generates corporate email accounts for new hires based on department selection.

It demonstrates core Object-Oriented Programming principles and basic input validation.

---

## 🚀 Features

- Department selection (Sales, Development, Accounting, None)
- Automatic corporate email generation
- Turkish character normalization for valid email formatting
- Secure random password generation (SecureRandom)
- Mailbox capacity management
- Alternate email configuration
- Encapsulation using getters and setters

---

## 🛠 Technologies Used

- Java
- OOP (Encapsulation, Constructors)
- SecureRandom (for password security)
- Scanner (CLI input)
- Git & GitHub
